Installationshinweise

Es gibt mehrere Möglichkeiten wie ich LaTeX auf eurem PC nutzen könnt. zum einen gibt es die Möglichkeit ein Packet-Manager und ein Editor zu installieren, und dann gibt es noch die Möglichkeit eine Online-Plattform zu nutzen. Das Gute an der Online-Plattform ist, ihr könnt nach der anmeldung direkt loslegen. Nachteil ist, ihr braucht immer Internet wenn ihr daran arbeiten wollt und oder Datenschutz. Ich will da auch nicht weiter darauf eingehen.

Da ihr, falls ihr es Lokal auf eurem Rechner nutzen wollt mehrere Sachen "beachten" müsst möchte ich euch das auch jetzt näher bringen.

Um Latex auf euren PC zu nutzen braucht ihr zum einen ein Packetmanager der, alle Packete und Dokumentenklassen zur verfügung stellt. Zum anderen braucht ihr noch ein Editor mit dem ihr die Dateien erstellt und der die Tex-Befehle umsetzt und euch ein Dokument erstellt.

🗄 Packet Manager​

Als erstes möchte ich euch drei Packetmanager näher bringen. Da währe zum einen MikTeX, TeXLive und MacTeX.

Die Installation der einzelnen habe ich auf den folgenden Seiten nochmal genauer beschrieben, hier soll euch nur eine Übersicht gegeben werden. Zum einen seht ihr, mit welchem Betriebsystem die jeweiligen kompatibel sind und zum andern hab ich euch den Link zur jeweiligen Downloadseite hinzugefügt.

MikTeX​

  • Betriebssystem: Windows / MacOS / Linux
  • Download-Link: https://miktex.org/download

TeXLive​

  • Betriebssystem: Windows / MacOS / Linux
  • Download-Link: https://www.tug.org/texlive/acquire-netinstall.html

MacTeX (TeXLive)​

  • Betriebssystem: MacOS
  • Download-Link: https://www.tug.org/mactex/mactex-download.html
tip

Ich selbst habe MikTeX auf Windows und MacTeX auf MacOS installiert und jeweils gute Erfahrungen gesammlt. MikTeX hatte bei mir auf MacOS Probleme verursacht, die ich selbst nicht lösen konnte. Daher seit nicht verzweifelt, falls einer der Packetmanager Probleme macht, veruscht einfach einen anderen aus.

📟 Editoren​

Als nächstes gibt es die Editoren, mit denen ihr schlussendlich die LaTeX-Dokumente erstellt.

Die Wahl hier bei ist Großteils Geschmacksache, da sie sich eigentlich nur in der Optik wirklich unterscheiden. Zu dieser Liste, die ich euch hier vorstelle, gibt es noch viele weitere. Die ihr gerne selbst ausprobieren dürft. Ich möchte es hier aber auf was wesentliche beschränken.

Bei der Online-Variante muss euch bewusst sein, dass ihr immer Internet zur Arbeit benötigt und bei der Visual Studio Code (vsCode) müsst ihr ein paar Dinge einstellen bevor ihr LaTeX schreiben könnt.

Falls ihr euch für MikTeX oder MacTeX entscheiden sollte, habt ihr auch einen vereinfachten Editor schon dabei. Diese habe ich selbst nie benutzt, daher kann ich nicht viel über deren Funktionsumfang sagen.

Meine Empfehlung für den Start ist TeXmaker oder TeXStudio.

TeXmaker​

  • Betriebssystem: Windows / MacOS / Linux
  • Download-Link: https://www.tug.org/mactex/mactex-download.html

TeXStudio​

  • Betriebssystem: Windows / MacOS / Linux
  • Download-Link: https://www.tug.org/mactex/mactex-download.html

Online​

  • Betriebssystem: unabhängig

Visual Studio Code​
